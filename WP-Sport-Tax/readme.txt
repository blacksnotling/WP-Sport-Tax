Add typical taxonomies used by sports sites - Team, Player, Competition, and Season

== Description ==

Creates a couple of default taxonomies for us on your Wordpress sites based on Sports. This includes:

*   Team
*   Player
*   League
*   Competition
*		Season



== Installation ==

1. Upload `WP-Sport-Tax` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
